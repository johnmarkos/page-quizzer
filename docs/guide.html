Getting Started with PageQuizzer

A guide for people who don't know what an API key is (yet)

PageQuizzer turns any web page into a quiz. You read something, click a button, and answer questions about it. That's it. The quiz questions are written by AI — specifically, Google's Gemini, which you can use for free with no credit card required. You just need to get a key that lets PageQuizzer talk to it. This guide walks you through that in about five minutes.

Step 1: Install PageQuizzer

  1. Go to the PageQuizzer page on the Chrome Web Store
  2. Click Add to Chrome
  3. Click Add extension when prompted

You'll see the PageQuizzer icon appear in your browser toolbar (top right). You may need to click the puzzle piece icon and pin it.

Step 2: Get your free Gemini API key

This is the only technical part. An "API key" is just a password that lets PageQuizzer use Google's AI on your behalf. Google gives these out for free.

  1. Go to aistudio.google.com/apikey
  2. Sign in with your Google account
  3. Click Create API key
  4. Click Create API key in new project
  5. A long string of letters and numbers will appear — that's your key. Click the copy button next to it.

Keep this tab open, or paste the key somewhere safe temporarily. You'll use it in the next step.

Step 3: Add your key to PageQuizzer

  1. Click the PageQuizzer icon in your Chrome toolbar
  2. Click the Settings gear icon
  3. Under AI Provider, select Google Gemini
  4. Paste your key into the API Key field
  5. Click Save

You're done. You only have to do this once.

Step 4: Quiz something

  1. Go to any article or web page you want to learn from
  2. Click the PageQuizzer icon
  3. Click Generate Quiz
  4. Answer the questions

That's the whole thing.

FAQ

Is it really free?
Yes. Google's free tier is generous for casual use. You'd have to quiz dozens of long articles every day before hitting any limits.

Is it safe?
Your API key is stored only in your browser. PageQuizzer never sends it anywhere except directly to Google. No accounts, no tracking.

What if something doesn't work?
File an issue at github.com/johnmarkos/page-quizzer/issues.
